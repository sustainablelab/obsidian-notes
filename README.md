# How to use it

- Clone this repository
- the `.obsidian` folder is in `doc/zet`:

    ```
    $ tree -a
    .
    ├── doc
    │   └── zet
    │       ├── .obsidian
    │       │   ├── app.json
    │       │   ├── appearance.json
    │       │   ├── core-plugins.json
    │       │   ├── graph.json
    │       │   ├── hotkeys.json
    │       │   ├── themes
    │       │   │   └── Obsidian gruvbox.css
    │       │   └── workspace
    ```

- Open the vault in obsidian

It opens as the graph. Click on a file node (a white dot) and it
looks like this:
